Descrição
A solicitação de pagamento indenizatório somenete poderá ser feita quando houve uma
solicitação expressa de servidor competente para entrega de um bem ou prestação de um
serviço fora de escopo contratual, desde que em razão de interesse público.

1.1.1.13 13. Analisar solicitação.
Descrição
Analisar o mérito da solicitação de pesquisa. Se for despesa decorrente de expiração de
contrato ou algo relacionado com contrato existente, justificar preço pelo contrato
respectivo. Se o objeto tiver relação com contrato existente com outra empresa ou
credenciamento válido, usar os mesmo para justificar preço.
Caso contrário, proceder à pesquisa em outra base para justificar o preço.

Subprocesso de saneamento despesa em caráter
indenizatório
2.1.1 Elementos do processo
2.1.1.1 T1. Identificar o motivo da inconformidade
Descrição
Analisar o mérito do motivo da inconformidade processual. Se for saneável (algo puder ser
feito legalmente para resolver), fazer tarefa seguinte. Caso contrário, arquivar o processo
informando ao interessado.
Executantes
Núcleo de Controle Interno
2.1.1.2 T2. Identificar o setor responsável pela correção
Descrição
De acordo com a tarefa a ser executada para saneamento do processo, identificar o setor
responsável por realizá-la, enviando os autos com solicitação de correção.
Executantes
Núcleo de Controle Interno
2.1.1.3 T3. Enviar o processo para correção
Descrição
Após identificação do setor e tarefa a ser realizada para correção do processo, fazer o
envio.
Executantes
Núcleo de Controle Interno
2.1.1.4 T4. Atribuir o processo para o servidor responsável
| 29
Descrição
Após identificar o motivo da inconformidade do processo, bem como a ação necessária
para corrigi-la, atribuir os autos ao servidor designado para executar a tarefa,
estabelecendo, de forma clara e objetiva, o que deve ser feito, como e quando.
2.1.1.5 T5. Executar a tarefa de saneamento
Descrição
Ao receber a designação para executar a tarefa necessária ao saneamento do processo,
adotar as medidas pertinentes para solução do caso, conforme orientação superior.
2.1.1.6 T6. Enviar o processo para demandante
Descrição
Concluída a correção das pendências processuais, enviar os autos para o setor demante
para prosseguimento do trâmite.
Informado
Núcleo de Controle Interno
2.1.1.7 T7. Conferir se a inconformidade foi saneada
Descrição
Analisar os autos observando se a correção solicitada na tarefa 2 foi realizada a contento.
Executantes
Núcleo de Controle Interno
